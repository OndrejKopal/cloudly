ProjectCloudly

ProjectCloudly is an enterprise grade IT Infrastructure and Applications Monitoring platform developed under the open source model.

Well, what does that mean?

Enterprise Grade: It was designed from “day one” to monitor tens of thousands to ultimately unlimited devices with a single instance. It brings the power, scalability and flexibility that enterprises and carriers demand. This is achieved thanks to the some of the coolest technologies in the industry (Hadoop, OpenTSDB, Django and such..)

Application Platform: While ProjectCloudly is useful “out of the box”, it is designed to be highly customizable and work with the industry standard Nagios to create an unique and integrated servers monitoring solution.

Open Source: ProjectCloudly is 100% Free and Open Source software, with no license fees, software subscriptions or special “enterprise” versions.

Features

Automatic Install
Automatic agent installation for any Linux; or alternative installer scripts/packages - works in any environment, really.

Automatic PLUGINS (NAGIOS COMPATIBLE)
Monitor anything and everything with 2000+ plugins built by our wonderful community of DevOps and Sysadmins. We're even compatible with Nagios plugins!
MongoDB, MySQL, Nginx, node.js, PostgreSQL, JVM, RabbitMQ, Redis, memcached, SQL Server, HAProxy ...
We will even take care of the complex plugin installation/deployment for you!

Advanced server monitoring alerts
Get notified of problems before they really affect your users - email, SMS, push notification, web hooks with integrated escalation management.

Graph your infrastructure
Investigate alerts with your own real-time graphs dashboards across servers and a multiple data centres.
Stack, filter and summarise all your data, kept forever so you can plan and spot trends.

Visibility across cloud providers
Automatically pull in metrics from Amazon Web Services, Google Cloud and Rackspace alongside other dedicated, on-premise or virtualised servers.
