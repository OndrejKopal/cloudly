ProjectCloudly - Smart, Smarter, the Smartest

Saves you money and shitload of resources!


ProjectCloudly is an enterprise grade Real-time IT Infrastructure and Applications Monitoring platform developed under the Open Source model.

Well, what does that mean?

Enterprise Grade: It was designed from “day one” to monitor tens of thousands to ultimately unlimited devices with a single instance. It brings the power, scalability and flexibility that enterprises and carriers demand. This is achieved thanks to the some of the coolest technologies ProjectCloudly utilises (Hadoop, OpenTSDB, Django and such..)

Application Platform: While ProjectCloudly is useful “out of the box”, it is designed to be highly customizable and work with the industry standard Nagios Plugins to create an unique and integrated servers monitoring solution. It does so fully automatically with very little or no interaction with the terminal prompt.

Open Source: ProjectCloudly is 100% Free and Open Source software, with no license fees, software subscriptions or special “enterprise” versions.

Features

Automatic Install
Automatic agent installation for any Unix like system (or Linux); or alternative installer scripts/packages that works in any environment, really.

Automatic PLUGINS (NAGIOS COMPATIBLE)
Monitor anything and everything with 2,000+ Nagios plugins!
MongoDB, MySQL, Nginx, node.js, PostgreSQL, JVM, RabbitMQ, Redis, memcached, SQL Server, HAProxy...
Our monitoring agent helps you and/or takes care of the complex plugin installation/deployment for you.

Advanced server monitoring alerts
Get notified of problems before they really affect your users - email, SMS, push notification, web hooks with integrated escalation management.

Graph your infrastructure
Investigate alerts with your own real-time graphs dashboards across servers and a multiple data centres.
Stack, filter and summarise all your data, kept for 30 days or forever so you can plan and spot trends.

Visibility across cloud providers
Automatically pull in metrics from Amazon Web Services, Digital Ocean, Google Cloud and Rackspace alongside other dedicated, on-premise or virtualised servers.
